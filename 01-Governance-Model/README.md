# 01 – Governance Model

This module defines the governance structure for managing cyber risk in a healthcare organization.

## Included

- RACI matrix for key roles (CISO, CIO, Privacy Officer, Clinical Leads, etc.).
- Decision-making model for risk acceptance, mitigation, and escalation.
- Control families aligned to NIST 800-53 and healthcare realities.
- Governance workflows (policy lifecycle, exception handling, and oversight).

## Objective

Provide a clear, repeatable governance structure that mid-sized healthcare organizations can adopt to improve accountability and decision-making around cyber risk.
