# 05 – Executive Dashboard (Power BI)

This module provides a Power BI dashboard design that visualizes cyber risk, human-factor indicators, and cloud security posture for healthcare leadership teams.

## Dashboard Components

### 1. PHI Exposure Heatmap
Shows risk levels across departments (ED, Outpatient, Billing, IT, etc.).

### 2. Human-Factor Risk Index
Combines:
- Training completion
- Reported incidents
- Fatigue indicators
- Workflow pressure

### 3. Cloud Security Posture Score
Based on:
- IAM findings
- GuardDuty alerts
- S3 bucket configurations
- Encryption coverage

### 4. Compliance Readiness Meter
Maps controls to:
- HIPAA
- PHIPA
- NIST CSF
- ISO 27001

### 5. Incident Response Maturity
Shows readiness across:
- Detection
- Containment
- Recovery
- Lessons learned

## Mock Dataset

A sample dataset will be included to support dashboard visuals:
- Department-level risk scores
- Human-factor modifiers
- Cloud security findings
- Compliance gaps

## Objective

Provide healthcare executives with a clear, visual understanding of cyber risk, enabling informed decision-making and prioritization.
